GIL: A parametrised intermediate language

Gillian uses an intermediate language for verification called GIL. It is parametric on actions of which the semantics is not predifined by the framework.

Focus on What Matters

We've worked hard on making Gillian as easy to use as possible. It comes with a shiny API that lets you describe your memory model and get a powerful tool with an elegant CLI for free.

Fully parametric meta-theoretical results

Gillian comes with fully parametric meta-theoretical results. We try to alleviate your work as much as possible: only prove what you implement, nothing more.
